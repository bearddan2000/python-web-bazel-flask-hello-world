# python-web-bazel-flask-hello-world

## Description
Creates a simple flask project.

## Tech stack
- bazel
- python
- flask

## Docker stack
- l.gcr.io/google/bazel:latest

## To run
`sudo ./install.sh -u`
http://localhost:5000

## To stop (optional)
`sudo ./install.sh -d`

## For help
`sudo ./install.sh -h`
